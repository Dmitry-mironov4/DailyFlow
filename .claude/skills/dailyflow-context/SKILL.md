# DailyFlow Project Context

## ⚠️ Источник истины

**`DailyFlow/CLAUDE.md`** — единственный источник истины по архитектуре, дизайну и решениям. Прочитай его в начале сессии и при любом сомнении.

## Жёсткие правила (нарушать нельзя)

1. **Только тёмная тема.** Никаких `Color.primary` без явного указания тёмного значения. Любые цвета — через токены из `Extensions/ColorExtensions.swift`.
2. **Только русский язык** в UI-строках. Любой английский текст в `Text(...)` — баг.
3. **Только `@Model` (SwiftData)** для сохраняемых данных. Не использовать `Codable` структуры в `UserDefaults` для бизнес-данных (только для настроек).
4. **Никаких сторонних SPM-зависимостей.** Только Apple-фреймворки.
5. **iOS 26+ минимум.** Не использовать обходы для старых iOS.
6. **View ≤ 150 строк.** Если ближе к лимиту — декомпозируй.
7. **`#Preview` обязателен** в каждом View с тёмной темой и реалистичными данными.
8. **MVVM «по необходимости».** Простой View не требует ViewModel — не плоди классы ради класса.

## Дизайн-токены (помни наизусть)

| Токен | HEX | Где |
|---|---|---|
| `bg.primary` | `#0D0D0D` | фон экрана |
| `bg.card` | `#1A1A1A` | фон карточки |
| `accent.teal` | `#2DD4A0` | задачи, экспорт |
| `accent.amber` | `#F0A23B` | привычки, стрики |
| `accent.purple` | `#9B8AE8` | настроение, стата |
| `text.primary` | `#F2F2F2` | основной текст |
| `text.secondary` | `#888888` | мета |
| `text.ghost` | `#666666` | плейсхолдеры |

**Запрещено:** градиенты, тени, blur, полупрозрачные оверлеи, эмодзи как иконки. Только SF Symbols и плоские цвета.

## Типографика — SF Pro

- **Title:** 21pt `.medium`
- **Body:** 13pt `.regular`
- **Caption:** 10pt, letter-spacing 0.5pt, ALL CAPS

## Раскладка

- Карточка: padding `.horizontal 16, .vertical 14`, corner radius `12`
- Между карточками: `12pt`
- Глобальный горизонтальный отступ экрана: `16pt`

## App Group

`group.com.dmitry.dailyflow` — для шаринга SwiftData store с виджетом.

## После любых изменений

- Запусти `/lint`

- Обнови раздел **Статус** и **Выполненные фичи** в `CLAUDE.md`
